# Arithmetic Operations

Addition
``` python
4 + 2
```

Subtraction
``` python
4 - 2
```

Multiplication
``` python
4 * 2
```

Division
``` python
4 / 2
```

Exponentiation (to the power of)
``` python
4 ** 2
```


## Examples

``` python
# Initialise variable
myMultiplication = 3 * 2

# Display result
print(myMultiplication)
```

``` python
# Initialise variable
mySquare = 3 ** 2

# Display result
print(mySquare)
```

``` python
# Initialise variable
myAge = 21

# Increment value
myAge = myAge + 1

# Display result
print(myAge)
```
